# DES240 5: Proposal

My experiments explored how to transform data into visual and interactive forms, with Experiment 3 (Real-Time Data) having the greatest impact on my research direction. Through using p5.js and APIs, I began to understand how data can transcend simple numerical representations to generate atmosphere, dynamism, and emotional qualities. This shifted my perspective from viewing data as merely a display object to seeing it as a resource that can actively construct experiences. I was drawn to the concept of interactive data design, especially in game systems, where user input and behavior can be collected and interpreted to shape personalized experiences. This aligns with the data humanism concepts discussed in the course and challenges traditional notions of data as purely objective or informative. I am particularly interested in how data can become subjective, responsive, and emotionally resonant.

Building on this foundation, my project will explore emotional states and experiences of self-expression in everyday life, focusing on how individuals process emotions they might not publicly express. I place this in a near-future scenario: digital systems (such as applications or game environments) will be able to continuously collect user interaction data and interpret behavioral patterns to infer users' emotional states. The system will not directly ask users about their feelings, but rather respond passively through interaction, creating a more atmospheric, indirect way to understand emotions. The data used in this project will combine user-generated interaction data (e.g., actions, choices, and engagement patterns), real-time environmental data (e.g., time and weather), and a dataset of simulated emotions for prototyping. This data will be dynamic and constantly evolving.

The system will not present raw data through charts or metrics; instead, it will interpret it into visual and interactive outputs, such as constantly changing atmospheres, generative particle systems, and responsive environments. Its goal is not to achieve precise representation, but to transform behavior into a perceptible experience. This approach relates to the concepts of data as matter and of interactive data systems, in which data is not only delivered but also actively shapes perception and interaction. This project may reveal how digital systems construct interpretations of human emotions and how these interpretations influence user experience.

The final result will be presented in an interactive, gamified, and visual form, where users can participate through simple input, and the system will respond instantly. Visually, it may include constantly evolving environments, abstract graphics, and dynamic motion systems that reflect underlying data patterns. This project aims to explore how data presents or misinterprets the human condition and to examine the relationship between personal experience and algorithmic interpretation. I hope viewers will feel the system's response to them and realize how their behavior is translated into data. Ideally, this will prompt a shift in perspective, moving away from viewing data as neutral information and instead recognizing it as a crucial factor in actively shaping our self-perception (data is not just numbers; it is experiences). 

This project is not intended to provide definitive answers, but rather to create a space for reflection, enabling users to perceive and question the relationship between data, identity, and emotion.
